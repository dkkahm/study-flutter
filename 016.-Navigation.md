### Basic
  - works as stack
  - works under MaterialApp hierachy

### Simple push and pop
  - push
```
onPressed: () {
    Navigator.push(
        context,
        MaterialPageRoute(
            builder: (context) {
                return ProductPage();
            },
        ),
    );
}
```
  - pop
```
onPressed: () {
    Navigator.pop(context);
}
```
  - pass data back
```
// push
Navigator.push<bool>(...)
    .then((value) {
        if (value != null && value) {
             _deleteProduct(index);
        }
    });

// pop
Navigator.pop<bool>(context, true);
```
